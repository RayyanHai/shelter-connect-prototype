# ShelterConnect Prototype

A centralized hub for homeless shelters and non-profits to coordinate resources and needs in real-time.

## Features
- **Real-time Feed:** View community offers and requests instantly.
- **Categorization:** Visual distinction between 'Offers' (Green) and 'Requests' (Red).
- **Responsive Design:** Built with Bootstrap 5 for mobile and desktop use.
